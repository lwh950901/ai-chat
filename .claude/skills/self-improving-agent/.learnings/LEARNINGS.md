## 2026-04-16: find-skills 和 self-improving-agent 使用范围

**发现:** 这两个技能不限于 OpenSpec 流程，适用于所有对话场景。

**when to use find-skills:**
- 开发中遇到需要某种功能时
- 询问"有没有能做 X 的技能"时
- 任何需要扩展能力的时候

**when to use self-improving-agent:**
- 命令执行失败时
- 用户纠正错误时
- 发现更好的方法时
- API 或工具调用失败时
- 任何错误或新发现
